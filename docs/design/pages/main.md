화면 중앙 상단에는 BizBeacon이라는 제목이 크고 진하게 표시되고, 그 아래에는 '비즈니스 인사이트 포털'이라는 부제가 작게 표시됩니다. 

화면 맨 좌측에는 과거 검색한 기록이 리스트로 보여야 하며 숨기거나 늘릴 수 있어야 합니다. 기록이 없는 경우 공백으로 두면 됩니다. 

화면 좌측 상단에는 dropdown/select 버튼이 하나 있고 직종을 선택할 수 있습니다. 직종의 종류는 전략기획, 연구개발, 영업, 마케팅, 인사, 재무, 기획, 총무, 디자인 기타입니다. 

직종 선택 밑에는 검색 창이 있는데 키워드나 웹사이트 주소를 입력 후 추가(+)버튼을 누르면 창이 하나 더 밑으로 생기고 엔터 버튼(enter)을 누르면 AI가 자동으로 찾아 줄 것입니다. 

검색 결과가 도출되면 직종 버튼 옆에 정량적 스코어(Business Impact Score)가 자동으로 표시됩니다. 문서로 요약된 검색 결과는 화면 가운데에 표시됩니다. 

화면 가운데 하단에는 검색 결과에 대한 페이지가 뜨는데 검색 전에는 공백이었다가 결과가 도출되면 1부터 N번까지 표시됩니다. 기본 10페이지이며 늘리기 버튼을 누르면 10페이지씩 늘릴 수 있습니다.
